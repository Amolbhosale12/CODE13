Looping in C language

Looping in C language is like setting up a repetitive task. You tell the computer to do something over and over again until a condition is met or until you say stop. It's like telling someone to count from 1 to 10, but instead of saying each number individually, you just tell them to "keep counting until you reach 10". That's what loops do in programmingâ€”they automate repetitive tasks by repeating a block of code until a condition is satisfied.
There are three types of loops in C as follow:

1.For Loop:
The for loop repeatedly executes a block of code until a specified condition evaluates to false.
Its syntax is as follows:

C
/*
for (initialization; condition; increment/decrement) {
code to be executed
}
*/
 #include <stdio.h>
int main() {
for (int i = 0; i < 5; i++) {
printf("Value of i: %d\n", i);
}
return 0;
}

2.While Loop:
The while loop executes a block of code as long as a specified condition is true.
Its syntax is as follows:

C
/*
while (condition) {
code to be executed
update condition
}
*/
 #include <stdio.h>
int main() {
int i = 0;
while (i < 5) {
printf("Value of i: %d\n", i);
i++;
}
return 0;
}

3.Do While Loop:
The do-while loop is similar to the while loop, but it executes the block of code at least once before checking the condition.
Its syntax is as follows:

C
/*
do {
code to be executed
update condition
} while (condition);
*/
#include <stdio.h>
int main() {
int i = 0;
do {
printf ("Value of i: %d\n", i);
i++;
} while (i < 5);
return 0;
}

Conditional statement in c language

Conditional statements in C language allow you to make decisions in your code based on certain conditions. They let your program choose between different paths of execution depending on whether a condition is true or false.
There are mainly four types of conditional statements in C:

1.if statement: The if statement executes a block of code if a specified condition is true.
Its syntax is:

C
/*
if (condition) {
code to be executed if condition is true
}
*/
 #include <stdio.h>
int main() {
int x = 10;
if (x > 5) {
printf("x is greater than 5\n");
}
return 0;
}

2.if-else statement: The if-else statement executes one block of code if the condition is true, and another block if the condition is false.
Its syntax is:

C
/*
if (condition) {
code to be executed if condition is true
} else {
code to be executed if condition is false
}
*/
 #include <stdio.h>
int main() {
int x = 3;
if (x % 2 == 0) {
printf("x is even\n");
} else {
printf("x is odd\n");
}
return 0;
}

3. Nested if statements: Nested if statements in C allow you to have one if or if-else statement inside another if or if-else block. This allows for more complex decision-making scenarios where you need to check multiple conditions.
Here's an example:

C
#include <stdio.h>
int main() {
int x = 10;
int y = 5;
if (x > 5) {
printf("x is greater than 5\n");
if (y > 2) {
printf("y is greater than 2\n");
} else {
printf("y is not greater than 2\n");
}
} else {
printf("x is not greater than 5\n");
}
return 0;
}

4.if-else ladder: The "if-else ladder" is an extension of the "if-else" construct in programming languages like C, C++, Java, and others. It allows you to test multiple conditions sequentially until one of them evaluates to true.
Here's the basic syntax:

C
/*
if (condition1) {
code to execute if condition1 is true<< /rb> } else if (condition2) {
code to execute if condition1 is false and condition2 is true
} else if (condition3) {

} else {
code to execute if all conditions are false
} */

#include <stdio.h>
