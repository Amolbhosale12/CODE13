Switch statement in C

The switch statement in C provides a way to execute different blocks of code based on the value of a variable. It's particularly useful when you have multiple conditions to check against a single variable.
Here's how it works:

C
#include <stdio.h>
int main() {
int choice;
printf("Enter a number between 1 and 3:");
scanf("%d", &choice);
switch (choice) {
case 1:
printf("You entered 1\n");
break;
case 2:
printf("You entered 2\n");
break;
case 3:
printf("You entered 3\n");
break;
default:
printf("Invalid choice\n");
}
return 0;
}

Each case label must be a constant expression.
â€¢ The break statement is used to exit the switch block as well as you can use with loops to exit.
â€¢ The default case is optional and is executed if none of the case values match the value of the variable being switched.

Recursion in C language

Recursion in C language is a programming technique where a function calls itself directly or indirectly to solve a problem. It's similar to a loop but achieved through function calls. Here's a basic example of a recursive function to calculate the factorial of a number.

C
#include <stdio.h>
// Recursive function to calculate factorial
 int factorial(int n) {
if(n == 0 || n == 1) {
return 1;
} else {
return n * factorial(n - 1);
}
}
int main() {
int num = 5;
printf("Factorial of %d is %d\n", num, factorial(num));
return 0;
}

Recursion can be a powerful and elegant solution for certain problems, but it's important to handle it carefully to avoid infinite recursion (calling the function infinitely) and excessive memory usage. Also, it may not always be the most efficient solution, especially for problems where iteration can be simpler and more efficient.

Array in c language

Arrays in C language are data structures that allow you to store multiple values of the same data type under one variable name. Each value in the array is accessed using an index, which starts from 0 and goes up to one less than the size of the array. Here's a basic example of how arrays are declared and used in C:

C
#include <stdio.h>
int main() {
// Declaring an array of integers with 5 elements
int numbers[5];
// Assigning values to elements of the array
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;
// Accessing and printing elements of the array
printf("Element 0: %d\n", numbers[0]);
printf("Element 1: %d\n", numbers[1]);
printf("Element 2: %d\n", numbers[2]);
printf("Element 3: %d\n", numbers[3]);
printf("Element 4: %d\n", numbers[4]);
return 0;
}

Data structures

In C, it's common to use structures (structs) or unions when you need to store different types of data within a single data structure. Here's how you can implement them:

C
#include <stdio.h>
//Define a struct to store information about a person
struct Person{
char name[50];
int age;
float height;
};
//Define a union to store either an integer or a float
union Number {
int integer;
float floating_point;
};
int main() {
//Example of using a struct
struct Person Person1;
strcpy(person1.name, "John");
person1.age = 30;
person1.height = 5.9;
printf("Person's name: %s\n", person1.name);
printf("Person's age: %d\n", person1.age);
printf("Person's height: %.2f\n", person1.height);
Example of using a union
union Number num;
num.integer = 10;
printf("Integer value: %d\n", num.integer);
num.floating_point = 3.14;
printf("Float value: %.2f\n", num.floating_point);
/*Notice that after assigning a float value to the union, the integer value is overwritten*/
printf("Integer value after assigning float: %d\n", num.integer);
return 0;
}

Union in c language

Unions in the C programming language allow you to store different types of data in the same memory location. Unlike structures (structs), where each member has its own memory location, in a union, all members share the same memory space. This means that a union variable can hold only one value at a time. Here's a basic example to illustrate unions in C:

C
#include <stdio.h>
//Define a struct to store information about a person
struct Person {
char name[50];
int age;
float height;
};
//Define a union to store either an integer or a float
 union Number {
int integer;
float floating_point;
};
int main() {
//Example of using a struct
struct Person person1;
strcpy(person1.name, "John");
person1.age = 30;
person1.height = 5.9;
printf("Person's name: %s\n", person1.name);
printf("Person's age: %d\n", person1.age);
printf("Person's height: %.2f\n", person1.height);
//Example of using a union
union Number num;
num.integer = 10;
printf("Integer value: %d\n", num.integer);
num.floating_point = 3.14;
printf("Float value: %.2f\n", num.floating_point);
/* Notice that after assigning a float value to the union, the integer value*/ is overwritten
printf("Integer value after assigning float: %d\n", num.integer);
return 0;
}

Storage Classes in C

In C programming, storage classes determine the lifetime, scope, and visibility of variables and functions within a program. C provides four storage classes:

1.Automatic Storage Class (auto): Variables declared within a function without any storage class specifier default to automatic storage class. They are created when the function is called and destroyed when the function exits. These variables have local scope and are not accessible outside the block where they are defined. auto is rarely used explicitly because it's the default behavior for local variables.

C
void exampleFunction() {
auto int x = 10; //auto is optional here
printf("%d\n", x);
}

2.Register Storage Class (register): Similar to automatic storage class but hints the compiler to store the variable in a CPU register for faster access. The use of register is merely a hint to the compiler; it may or may not comply. Cannot take the address of a register variable. As with auto, register is rarely used explicitly because compilers are typically efficient at optimizing variable storage. c

C
void exampleFunction() {
register int y = 20;
printf("%d\n", y);
}

3.Static Storage Class (static): Variables declared with static storage class retain their value between function calls. They have a lifespan throughout the entire program execution. static variables declared within a function are initialized only once and retain their value between function calls. static variables declared at the file scope (outside of any function) are accessible only within the file in which they are declared.

C
void exampleFunction() {
static int z = 30;
printf("%d\n", z);
z++;
}

4.External or Global Storage Class (extern): Variables declared with extern storage class are defined outside of any function, typically at the top of the file or in header files. They are visible to all functions within the file or, if declared in a header file, to all files that include that header. When declared extern, the variable is only declared, not initialized. The actual initialization happens elsewhere.

C
//File1.c
int globalVar = 40;
//File2.c
extern int globalVar; //Declaration of globalVar from File1.c
void exampleFunction() {
printf("%d\n", globalVar);
}
