Pointer

In C language, a pointer is a variable that stores the memory address of another variable. Pointers are widely used in C for various purposes such as dynamic memory allocation, accessing array elements, and implementing data structures like linked lists and trees.

C
#include <stdio.h>
int main(){
int value=10;
int *ptr;
ptr= &value;
printf("%d",*ptr);
return 0;
}
